# SUIKA-GAME (フルーツマージゲーム)

## 概要
SUIKA-GAMEは、ブラウザで手軽に楽しめる物理ベースのフルーツマージパズルゲームです。プレイヤーは様々なフルーツを落とし、同じ種類のフルーツをぶつけることでより大きなフルーツへと進化させていきます。コンテナからフルーツがあふれないように注意しながら、ハイスコアを目指したり、クリアタイムを競ったり、特別なモードで遊んだりすることができます。

## 遊び方（操作方法）
1.  **ゲーム開始:** メインメニューから遊びたいモードを選択してください。
2.  **フルーツを落とす:** マウスカーソルを動かすことで、次に落ちるフルーツの位置を調整できます。クリック（またはタップ）することで、その位置にフルーツが落下します。
3.  **フルーツをマージする:** 同じ種類のフルーツ同士がぶつかると、より大きな次の種類のフルーツへと合体します。
4.  **ゲームオーバー:** フルーツがコンテナの最上部（危険ゾーン）からあふれそうになると、ゲームオーバーになります。

### 各ゲームモード
*   **通常モード:** フルーツを合体させてスコアを稼ぎ、ハイスコアを目指すエンドレスモードです。
*   **タイムアタック:** 最大レベルのフルーツ（桃色のスイカ）を2つマージして消すまでの時間を競います。クリアタイムが表示されます。
*   **接待モード（隠し）:** 落ちてくるフルーツの種類が徐々に大きくなっていきます。最終的には大きなフルーツばかり降ってくるサービスモード。スコアは100倍になります。
*   **虐待モード（隠し）:** 赤色、黄色、青緑色のフルーツの出現頻度が増加し、オレンジ色、黄緑色のフルーツは出現しません。より難易度の高いモードです。

### 隠しモードの出し方
メインメニュー画面で10回クリックすると、隠しモードのボタンが出現します。

## 工夫した点
*   **Matter.js物理エンジン:** フルーツのリアルな落下、衝突、マージの挙動を実現するために、高性能な2D物理エンジンMatter.jsを採用しました。これにより、予測不能で楽しいゲーム体験を提供します。
*   **多様なゲームモード:** プレイヤーの好みに合わせて選べる「通常モード」「タイムアタック」に加え、遊び心をくすぐる「接待モード」「虐待モード」といったユニークな隠しモードを実装しました。
*   **堅牢なゲームオーバー/勝利条件:** フルーツが危険ゾーンに一定時間留まることでゲームオーバーとなるセンサーベースの判定を導入し、意図しないゲーム終了を防ぎました。タイムアタックモードでは、特定の条件達成で勝利画面が表示されます。
*   **モードに応じたゲームバランス:** 接待モードではフルーツの出現レベルの調整とスコア倍率、虐待モードでは特定のフルーツの出現確率調整を行うことで、各モードで異なる戦略と楽しさを提供します。
*   **シングルHTMLファイルでの完結:** 複雑なビルドプロセスなしに、単一のHTMLファイルを開くだけでゲームがプレイできるよう設計しました。これにより、手軽な共有とデプロイが可能です。
